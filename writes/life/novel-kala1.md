# 等一个电话

“喔哦，怪你过分美丽......” “哔......哔......哔.......哔...... 哔......”

张国荣深情的歌声与公交车刷卡的声音同时响起，各干各的，一点也不尊重对方。

卡拉伸手掏出一部滑盖手机，屏幕上闪烁着一串生硬数字，他仅仅撇了一眼就按下了红键。

“我就知道”卡拉心想，“肯定是电话诈骗。”

这已经是第九十九次了，卡拉绝不会记错。自从换了号码到现在，他一直在想号码簿中谁会第一个打电话来呢 ？

“喂，谁啊？”

“你大爷，在干嘛呢！”

“有事说事，没事滚蛋。”
······

“喂，美女，咋这么有空找我啊？”

“呵呵，想你喽，在干嘛啊。”

“写诗呢。”

“哇，好厉害啊。”
······

以上对话或者短信都只存在于他的幻想中。他一开始坚信着，然后期待着，之后抱有一丝希望，最后连一线可能都被他抛弃了。在接到连续三十一次诈骗电话和短信后，卡拉就对陌生的号码深恶痛绝，每次遇上，直接关闭。至今，尚无人问候卡拉。

“卡拉啊！你就是一条赤裸裸的屌丝呵，连手机也被牵连着寂寞无垠啊。” ——《卡拉日记》

现在正是太阳下山、草民归巢、公路堵塞的时间段，卡拉不曾奢望过座位，径直走向末尾，抓杆站稳。举起的双手像是在投降。站着的人向坐着的人投降，公交车向小轿车投降，人们想生活投降。

不知是黑夜追赶汽车，还是汽车驶向黑夜，反正，天它就黑了，路灯它就亮了，高楼大厦它就璀璨了，卡拉也不知怎的就伤感了。他想啊，下辈子就不能这样了。投个好人家，不要再被人遗弃了，上学的时候认真读书，考它个清华北大，都多面子啊！大学的时候不能一个人闷着，应该找些推心置腹的好友，大口喝酒，大块吃肉，多潇洒啊！毕业了就不要去找工作了，写一部精彩绝伦的小说，再拿去画漫画，做游戏，拍电影，名利双收，多让人羡慕啊！赚的钱就捐一部分改善交通好了······下辈子，一定要好好珍惜拥有的美好，一定要。下辈子······卡拉此时觉得自己老了。但他才刚从大学毕业一年啊，人生路漫漫，下辈子遥遥无期。

“唉······”幻想破灭的卡拉忍不住叹了口气。马上又意识到自己在公共场所，不方便大声喧哗，有点讪讪地看了看周围的人。坐着的在玩手机，站着的也在玩手机。手机，远比卡拉这个活人重要，哪怕他异常地叹了口气，卡拉幻想中好奇的眼光一丝也不曾存在，只有他旁边的大叔不动声色与他拉开了一点距离。卡拉失望了，他总是不自禁地失望。

公交车走走停停，卡拉无聊地数起了窗外的路灯。
······

“卡拉永远 OK !"谁的电话铃声响起。

这是的深圳卫视节目"年代秀"里的一句话，卡拉看过好几次这个节目，他最恨这句话了，每次听到，都想跑到那胖胖的主持人面前，给他讲一个卡拉与 OK 的悲剧，然后狠狠骂他妖言惑众。这句话是在热辣辣地讽刺他啊，卡拉觉得。

卡拉已经知道声音的源头在哪里了，原来他旁边的大叔在他夜观星像时已经换成了 一位女孩子了，女孩背对着他的左侧，若卡拉面向车头的话，女孩的头顶就在卡拉的鼻子面前。卡拉的目光马上就被女孩的马尾吸引了，马尾又长又直，乌黑发亮，以浅蓝色的 T 裇为背景，宛如碧海蓝天间跃然而起的一尾墨黑鲤鱼。

“所谓，飘柔就是这样自信，也不过如此吧！”卡拉心想。

多么温馨的马尾啊，初中经常坐卡拉前面的班花也有一条骄傲的马尾，卡拉在上课时总是忍不住要看着它发呆，卡拉觉得这么柔顺的头发握在手里一定舒服极了，有很多次他想伸出手摸摸，又觉得她一定会生气的，同学一定会笑话他的，蠢蠢欲动的手被自卑与面子死死钉住，只隔了三四厘米的空气啊。 最后卡拉只能眼巴巴看着马尾一跳一跳地去了另一所高中，不带走一片云彩······

卡拉又看到眼前女孩那绑马尾的绳子上还有一个装饰，椭圆形的，上面的英文字母是，"OK"? 卡拉笑了，是回应 O 与 K 之间的笑脸表情吗？还是说他想起了他的小笔友？

卡拉曾经有也只有一位笔友，是个女孩，她本来是卡拉的小学同桌，后来转学了，但两人一直保持通信，无话不说，是纸上的青梅竹马。也许正是如此，卡拉讨厌其他联系方式，对书信情有独钟。可惜，只有笔名为“OK”的女孩理解卡拉的爱好，其他同学都觉得卡拉老土，这也许是卡拉一直无法融入身边世界的原因之一吧。后来，笔友敌不过女友，两年前，当卡拉第一次向 OK 炫耀他的爱情时，女孩只回了三个字——我等你，然后了无音信，彻底从卡拉的世界消失。卡拉当时那一个悔啊，仿佛心里多了一个空洞，凉飕飕的，苦溜溜的，怎么也无法填补，他反而希望自己从没有谈过恋爱，比起女友，卡拉更舍不得笔友，这个笔友已经陪伴了他十几年了，已经是卡拉心田里弥漫的草原了。“唉！人总要等到失去以后才懂得珍惜”卡拉当时心里暗叹道。后来，卡拉每次见到女友都会想起”逝去“的笔友，心里不是滋味，只好分手了。现在，卡拉只剩下自己了。

“生活就是拿这样东西去换那样东西，直到一无所有。”——《卡拉日记》

公交车上挺拥挤的，卡拉死死抓紧扶手，怕因为汽车颠簸，唐突佳人。尽管如此，女孩的发梢还是会时不时挑逗他的神经，丝丝痒痒的。卡拉突然有一股很强的冲动，他忍不住微微低头，嗅了一下。“好淡啊。”卡拉心想，微弱的香气几乎没什么味道，但他就喜欢这种朴素的感觉。卡拉又将头转向车窗，玻璃上隐约显现着女孩模糊的侧脸，看不清楚，卡拉觉得没啥意思，他还是更喜欢头发的味道，又转过了头。让卡拉吃惊的是，女孩忽然要转过身来。“被发现了？”卡拉的心都跳到了大脑里，咚咚直响，他有点做贼心虚地向后仰了仰身子。

“那个，你好······我可以借一下你的手机吗？”女孩有点脸红地看着卡拉。

“啊！？”卡拉明显呆愣了一下，他有点反应不过来，警察叔叔贴的告示上都说了，不要借陌生人手机！这个女孩，刚从火星上来？

“我的，没电了，······”女孩欲言又止。

此刻，卡拉心里有两只嘴巴在争执着。

“借了吧，说不定人家有急事呢？而且是个美女呦，或许会感激你，交个朋友啊·····”

“千万别上当，这人肯定是个骗子，别看她长得漂亮，你忘了张无忌他妈说的了，越是漂亮的女人越会骗人······”

结果，卡拉还是递给了女孩他的手机，反正，屌丝的手机就是拿来当闹钟用的，卡拉这么想。其实，他心里还有另一种想法——被骗也总比被无视好。卡拉觉得自己快没得救了。

女孩接过手机，道了声谢谢，然后很快发了一条短信，就把手机还给了卡拉，女孩有道了一声谢，笑得两眼眯起，有点狡黠，在卡拉眼里和动漫中穿水手服的妹子一样可爱。

卡啦其实很想搭讪几句的，千金难得，美女更难求，能在六十多亿茫茫人海中有所瓜葛，就是缘分吧！也许对方就是自己命中注定的另一半呢······卡拉幸福地幻想着，忘了说话了，看着女孩转过身去······卡拉觉得自己真是个羞涩的男人。

“七夕桥，到了，请乘客从后门下车，开门请注意······”

七夕桥正是卡拉所住的地方， 是一个城中村，一个职场菜鸟的聚居地。车上大多数年轻男女的终点都是这里。那漂亮女孩也在这儿下车了，随人流飘走，渐渐离开了卡拉的视野。卡拉收回目光，低头走向另一个方向。“又是无聊的一天啊，这日子啥时候到头啊！唉！”卡拉的心里很郁闷，特别是脑海中依然晃动着一根马尾，他记得 OK 说她也是扎着马尾的，一定很好看。卡拉觉得，也许回去该把一箱子信拿出来重温一下了，在无数个失眠的夜里，他都是这么做的，之后他就会做个好梦，梦中他追逐着一根俏皮的马尾，直到春暖花开，直到与子偕老。 卡拉不得不承认他想 OK 了，很想很想，他想啊，要是两个人能在一起该有多好啊······

“怪我过分着迷······”张国荣深情的歌声又响起了。卡拉掏出手机，有一条新短信，又是一串冷涩的数字。第一百次了，卡拉有种摔手机的冲动······

“算了，第一百次啊，值得纪念。”卡拉打开了短信，本以为的“你已被某某节目抽中为幸运嘉宾快来领奖”之类的文字没有出现，上面反而写着：卡拉，你还是那么笨啊！不懂和女孩子搭讪。

卡拉突然就停下脚步了······卡拉突然就转头飞奔了······他已经很久没这样跑过了，很久没这样撞人了，很久没这样闯红灯了，卡拉此时感觉老天爷对他真好。

卡拉很快就跑回了公交车站，气喘吁吁。他又停下脚步了，他始终望着那个方向，出神。就像古时躲在墙角的穷酸书生，眼睁睁看着大小姐出嫁。卡拉突然不知道自己该去何方，他只是卡拉啊，爱幻想的卡拉啊 ，不 OK 的卡拉啊。卡拉转身要回去了，他决定把宿舍里所有的泡面都煮了吃，一直吃，吃到昏迷不醒就最好了，明天醒来就去上班，然后下班，然后上班，然后下班·····

"嘿！你是在找我吗？"有人从后面拍了一下卡拉的肩膀，是一个漂亮女孩，眯着眼，那笑脸就像一只奸计得逞的小狐狸 。

“你是 OK？”卡拉凝视着对方的眼眸。

“你是卡拉我就是 OK。”女孩微仰着头，也看着卡拉，依旧笑靥如花，让昏黄路灯黯然无光。

她是 OK，只有 OK 知道卡拉的笔名是卡拉，卡拉只在写信或写日记时叫卡拉。

“这剧情会不会太狗血了。”卡拉不自禁笑了，不自禁。

“管它呢，喜欢就好啦！”
······

”人生就是得到又失去，失去又得到，最后剩下喜欢的。“——《卡拉日记》
